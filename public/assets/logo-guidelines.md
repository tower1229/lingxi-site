# LingXi Character Logo Usage

## Files
- `assets/logo.svg`: default icon with white background and black structure
- `assets/logo-mark.svg`: transparent character mark (`LX`) with red micro-accent
- `assets/logo-primary.svg`: website header logo (`LX` + `LINGXI`)
- `assets/logo-mono.svg`: monochrome mark using `currentColor`

## Recommended usage
- Website navbar: `assets/logo-primary.svg`
- Favicon/app icon: `assets/logo.svg`
- Single-color areas: `assets/logo-mono.svg`

## Spacing and size
- Minimum display size (mark): `18px`
- Minimum display size (primary): `120px`
- Clear space around logo: at least the width of the `L` vertical stroke

## Color notes
- Primary white: `#FFFFFF`
- Secondary black: `#111111`
- Accent red (minimal use): `#D90429`
